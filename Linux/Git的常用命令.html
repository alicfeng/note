Git的常用命令

准备事项为本地创建一个git仓库。

从远程数据库克隆数据

git clone <仓库代码的http链接>

执行status命令以确认tutorial目录的状态。

git status

将文件加入到索引，要使用add命令。在指定加入索引的文件。用空格分割可以指定多个文件。

git add <file>

指定参数「.」，可以把所有的文件加入到索引

git add .

既然文件已加入到索引，我们就可以提交文件了。请执行如下显示的commit命令。

git commit -m "更新文件的说明"

将文件推送到远程分支数据库

git push origin master
