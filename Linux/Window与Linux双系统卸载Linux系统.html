Window与Linux双系统卸载Linux系统

卸载Linux系统的方法：

1：先下载Mbrfix这个软件

2：下载之后，将其解压到你的系统目录（一般是c:\）

3：如果不在C盘，那就直接拖到C:Windows:systeam32下，用起来方便

4：打开windows命令行（开始/附件/命令行）（以管理员身份运行）或者打开的时候按住Shift+Ctrl

5:键入： cd \ [按回车] mbrfix /drive 0 fixmbr /yes [按回车]，这步是删除Grub引导程序的

6：然后重启看看，启动是不是直接就时win7

7如果是直接启动win7，那就删除Linux系统的那个盘，我的电脑——管理——磁盘管理——Linux系统所在盘，直接删除

8：如果不是重复4-6步骤，直到删除那个引导程序

一般情况下，一次就可以删除那个Grub引导程序的
